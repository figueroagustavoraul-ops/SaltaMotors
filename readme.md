# Salta Motors - Landing Page Automatizada con Asistente Comercial IA

Este repositorio contiene una maqueta funcional de alta conversión orientada a la industria automotriz (Concesionarias), diseñada específicamente para solucionar la pérdida de clientes fuera del horario comercial mediante interfaces reactivas de atención automatizada.

## 🚀 Arquitectura Técnica y Tecnologías Utilizadas

El proyecto fue desarrollado bajo una arquitectura desacoplada de código limpio, separando la capa de presentación de la lógica del cliente:

* **HTML5 Semántico:** Implementación de etiquetas estructurales puras (`<header>`, `<main>`, `<section>`, `<article>`) optimizadas para accesibilidad y posicionamiento orgánico en motores de búsqueda (SEO local).
* **CSS3 Custom Properties (Variables Modulares):** Maquetación responsiva a través de sistemas de rejillas estables (`CSS Grid`) y cajas de alineación (`Flexbox`). Modulación de paleta de colores corporativa premium escalable mediante propiedades raíz (`:root`).
* **Vanilla JavaScript (ES6+):** Programación pura orientada a eventos del lado del cliente. Control dinámico del DOM para la inyección de nodos asíncronos y sistema conversacional basado en mapeo de intenciones y descriptores de palabras clave mediante estructuras condicionales lógicas de procesamiento de texto.

## 🤖 El Valor de Negocio (Solución Propuesta)
Las PyMEs locales carecen habitualmente de canales de respuesta inmediata nocturnos. Esta interfaz integra un Widget reactivo que emula el comportamiento de una API Conversacional, reteniendo la intención de compra del usuario mediante flujos interactivos de información sobre precios, requisitos legales y planes de créditos bancarios, cerrando el ciclo con la captura de leads (contactos).
